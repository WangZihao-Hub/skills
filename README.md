# Claude Code Skills

克隆到 `~/.claude/skills/` 即可使用：

```bash
git clone git@github.com:WangZihao-Hub/skills.git ~/.claude/skills/
```

## 技能列表

| 技能 | 说明 |
|------|------|
| **brainstorming** | 创意探索与需求分析 |
| **docx** | Word 文档处理 |
| **find-skills** | 查找可用技能 |
| **frontend-design** | 前端设计 |
| **guizang-ppt-skill** | PPT 生成 |
| **markitdown** | 文件格式转换 |
| **obsidian-skills** | Obsidian 笔记处理 |
| **paper-interpreter** | 论文深度解读，生成 HTML 报告 |
| **paper-pages** | 论文分页处理 |
| **pdf** | PDF 处理（提取、合并、表单、OCR 等） |
| **planning-with-files-zh** | 文件规划 |
| **playwright-skill** | 浏览器自动化 |
| **pptx** | PPTX 处理 |
| **publish-html-to-pages** | HTML 发布为 GitHub Pages |
| **skill-creator** | 创建/编辑技能 |
| **using-superpowers** | 超级能力使用 |
| **xlsx** | Excel 处理 |
